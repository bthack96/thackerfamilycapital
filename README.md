# Thacker Family Capital - Investment Blog

A clean, minimal blog for sharing investment insights and financial analysis.

## Features

- Search functionality
- Category filtering (Stocks, Crypto, ETFs, Real Estate, Retirement)
- Responsive design
- Fast and lightweight

## How to Add New Posts

1. Copy `post-template.html` and rename it (e.g., `my-new-post.html`)
2. Edit the content, title, category, and date in your new file
3. Add the post information to `posts-data.js`:

```javascript
{
    id: 7,
    title: "Your Post Title",
    excerpt: "A brief description of your post...",
    category: "stocks", // options: stocks, crypto, etfs, real-estate, retirement
    date: "January 21, 2026",
    readTime: "5 min read"
}
```

4. Commit and push your changes to GitHub

## Local Development

Simply open `index.html` in your web browser to view the site locally.
